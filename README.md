## 🗒️ 스터디 / 미션 방식

#### 1. 개인 알고리즘 스터디 방식
- 본인의 '이름' 폴더를 생성하여 알고리즘 문제 풀이한 소스 코드 파일들 자유롭게 업로드

<br>

#### 2. 단체 알고리즘 미션 방식
- `매주 월요일` 새로운 미션이 올라옵니다.
- `🥇 난이도 上 문제 1개`, `난이도 中 기본 문제 3개`가 업로드 됩니다.
- 미션을 해결하고 `일요일 저녁`까지 본인의 폴더 내, '미션' > '날짜-주차' 형식의 폴더에 코드들을 업로드 합니다. (ex :'2021-08-02주차')
