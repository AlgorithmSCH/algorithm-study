# 문제-1676 팩토리얼 0의 개수

- 수학 문제이다.
- n 팩토리얼을 계산했을 때, 끝에 있는 0의 개수를 출력하는 프로그램이다.
- 해당 문제는 n보다 작은 5의 제곱마다 나눠주어 더한 값이 답이었다.
```C
int main() {
    int ans = 0;

    int n;
    cin >> n;

    for (int i = 5; i <= n; i *= 5)
        ans += n / i;

    cout << ans << '\n';
}
```

- 대충 규칙만 찾으면 쉬운데
- 그 규칙을 찾기가 더럽게 힘들다;;
